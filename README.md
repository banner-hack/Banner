# AI.pic



## 製品概要
『AI.pic』は部屋の雰囲気にマッチした利用者好みの絵を部屋の写真を撮影・好みの絵を選択することで簡単に作成できるwebサービスです。

製品ページ：https://jphack.vercel.app/

スマートフォンで起動してください。
カメラが起動しない場合、お使いのスマートフォンのカメラの設定を変更してください。
